# Mini ERP (Work In Progress)

## 🎯 Project Goal
This is a personal project focused on building a foundational ERP system using ASP.NET Core MVC and Entity Framework Core.

The project emphasizes backend business logic implementation, including employee management, department management, and leave request workflows, while maintaining a clean and scalable architecture.

Current focus areas:
- Business workflow implementation
- Service layer architecture
- Entity relationships and validation
- Soft delete handling
- Result pattern for error handling
- Maintainable MVC structure

---

## 🚀 Tech Stack
- Framework: ASP.NET Core MVC (.NET 10)
- Database: SQL Server
- ORM: Entity Framework Core (EF Core)
- Architecture: MVC + Service Layer Architecture
- Frontend: Razor Views
